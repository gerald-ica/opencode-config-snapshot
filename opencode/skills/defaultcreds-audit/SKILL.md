# Default Credentials Audit

Use this skill for:

- `/Users/wendyly/.local/share/agent-skills/external/DefaultCreds-cheat-sheet`

## Scope

- This repo is a defensive reference for default credential discovery and remediation.
- Use it to harden owned systems and validate insecure defaults during authorized assessments.

## Default Workflow

1. Treat the repo primarily as a reference dataset.
2. Inspect `README.md`, `requirements.txt`, and the `creds` script before installing.
3. Prefer targeted lookup and remediation guidance over broad automation.
4. Record any findings as sensitive security issues, not routine notes.

## Safety

- Do not use the dataset for unauthorized access.
- Sanitize exported reports before sharing them outside the security context.
